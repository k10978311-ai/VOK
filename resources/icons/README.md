# Platform Icons

This folder contains icons for various social media platforms used in the downloader.

## Supported Platforms

Place PNG icons (preferably 64x64 or higher) with these exact names:

- `youtube.png` - YouTube
- `tiktok.png` - TikTok  
- `douyin.png` - Douyin
- `kuaishou.png` - Kuaishou
- `instagram.png` - Instagram
- `facebook.png` - Facebook
- `pinterest.png` - Pinterest
- `twitter.png` - X (Twitter)

## Fallback

If an icon file is not found, the app will display a colored badge with the platform's first letter:
- YouTube: Red (Y)
- TikTok: Black (T)
- Instagram: Pink (I)
- Facebook: Blue (F)
- Pinterest: Red (P)
- Twitter: Blue (T)

## Icon Guidelines

- **Format**: PNG with transparency
- **Size**: 64x64 pixels (or higher, will be scaled down)
- **Style**: Simple, recognizable platform logos
- **License**: Ensure you have the right to use platform logos

## Where to Get Icons

You can download official brand assets from:
- YouTube: https://www.youtube.com/about/brand-resources/
- TikTok: https://newsroom.tiktok.com/en-us/brand-guidelines
- Instagram: https://about.meta.com/brand/resources/instagram/
- Facebook: https://about.meta.com/brand/resources/facebook/
- Pinterest: https://business.pinterest.com/brand-guidelines/
